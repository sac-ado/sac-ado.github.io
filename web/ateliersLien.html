Liens entre les différents ateliers

Plusieurs enchaînements d'ateliers sont possibles. Cependant, il est préférable de faire certains ateliers avant dautres.

Afin de vous aider dans vos choix, vous trouverez ci-dessous un schéma représentant les liens entre les différents ateliers. Les flèches pleines représentent la nécessité d'effectuer un atelier avant un autre ; tandis que la flèche en pointillé indique simplement qu'il est conseillé de faire un atelier avant un autre.

Sur la gauche, vous trouverez les liens entre les différents thèmes et, sur la droite, les liens entre les différents ateliers.
