Liens entre les différents ateliers

Plusieurs enchaînements d'ateliers sont possibles. Cependant, il est préférable de faire certains ateliers avant dautres.

Afin de vous aider dans vos choix, vous trouverez ci-dessous un schéma représentant les liens entre les différents ateliers. Vous n'êtes pas obligé de suivre toutes les indications

La flèche du milieu indique l'ordre conseillé entre les différents thèmes ; les autres flèches indiquent les liens entre les ateliers d'un même thème.
